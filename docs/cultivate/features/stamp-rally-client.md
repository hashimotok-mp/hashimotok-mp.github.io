## <:a003:1354004444316176495> 概要

16Fes内で実施する**交流スタンプラリー**用BOTです。

参加者が各MBTIロール所持者と指定VC内で5分以上交流すると、そのMBTIのスタンプを獲得します。

スタンプはMBTIごとに1回のみ取得可能です。

ユーザーごとに「どのMBTIと交流済みか」「誰と達成したか」を確認できます。

> 本BOTはMBTIに限らず、任意のロールグループで汎用的に利用可能です。

運営が指定した**カテゴリー**内のVCのみ計測します。

```
📁 MBTI交流広場（指定カテゴリ）
 ┣ ✅ MBTI新規開拓
 ┣ ✅ 新規開拓時限式
 ┣ ✅ 新規開拓
 ┣ ✅ フリル
 （増える可能性あり）

📁 雑談（非指定）
 ┣ ❌ general（計測されない）
```

> 指定カテゴリー外のVCは一切計測対象外です。

## 🎯 スタンプ獲得条件

### 基本条件（以下すべて満たした場合のみ計測開始）

- 指定カテゴリー内のVCであること
- VCに自分以外の参加者が1名以上いること
- 自身がMBTIロールを所持していること
- 相手がMBTIロールを所持していること
- BOTではないこと
- AFKチャンネルではないこと

### 取得条件

対象MBTIのユーザーと **5分（300秒）** 以上同席でスタンプ獲得

> 交流時間は管理者が設定変更可能です。

### スタンプ取得ルール

**■ 1グループ1回のみ**
同じMBTIタイプのスタンプは1回だけ取得可能

**■ 重複取得不可**
取得済みのMBTIは再計測しない

**■ 複数同時進行**
複数の未取得MBTIと同時に交流可能（全ペア同時計測）

**■ 自分自身は対象外**
自分が持つMBTIロールと同じタイプは、他の人との同席が必要

**■ 切断でリセット**
VC退出で未達成の経過時間はリセット

**■ ペアごとに独立**
片方だけ退出しても他方の計測は継続

### 取得例

```
自分：ESTJ
相手：INFP

↓ 5分以上同席

INFPスタンプ 獲得 ✅

（別のINFPと交流しても追加取得はなし）
```

## 🖼️ スタンプカード

### 表示方法

**ボタン**（推奨）またはスラッシュコマンド `/スタンプカード` で表示します。

> サーバー設定でコマンドの有効/無効を切替可能です。

### 画像生成（有効時）

運営が用意した素材を使ってスタンプカード画像を自動生成します。

- **台紙画像**：スタンプカードの背景（サーバー側で用意）
- **スタンプ画像**：MBTIタイプごとの取得済みスタンプ画像
- **未取得画像**：未取得スタンプ用の空欄画像

各スタンプ画像の配置位置は**座標（x, y）**で指定可能です。

## 📢 通知機能

### スタンプ獲得ログ

ユーザーがスタンプを獲得すると、**運営ログチャンネル**に自動通知されます。

```
【スタンプ獲得】🎉
ユーザー：@user
獲得MBTI：INFP 🦋
交流相手：@partner
達成数：7 / 16
```

> 本人への通知は**ログチャンネルが設定されていればON、未設定ならOFF**です。

### コンプリート通知

全16タイプを達成すると、特別通知が送信されます。

```
🏆 交流スタンプラリー完全制覇！
🎊 @user が全16タイプとの交流を達成しました！
```

> ロール付与は**設定すればON、未設定ならOFF**となります。

## ✅ 決定事項

- **画像素材**：運営側で用意（台紙・スタンプ画像）
- **スタンプ獲得時の本人通知**：ログチャンネル設定時はON、未設定時はOFF
- **コンプリート特典ロール付与**：設定時はON、未設定時はOFF

## 🛡️ 不正防止・制約

1. BOTアカウントは計測から完全除外
2. AFKチャンネル在籍中は計測停止
3. MBTIロール未所持者は参加不可
4. 指定カテゴリー外のVCは無効
5. 同一MBTIのスタンプ重複取得不可
6. 自分自身は計測対象外
7. VC人数が1名になったら全計測を停止
8. 取得済みMBTIは再計測しない
9. VC退出で未達成の経過時間はリセット

---

## ⚙️ 設定パネル

管理者は設定パネルから以下を設定できます。

- **有効/無効**：機能全体のON/OFF
- **対象カテゴリー**：計測対象のカテゴリーを指定
- **スタンプグループ**：スタンプの種類と対応ロールを設定
- **交流時間**：スタンプ獲得に必要な同席時間（デフォルト5分）
- **ログチャンネル**：獲得通知の送信先
- **獲得通知**：ON/OFF・通知方式（DM/チャンネル）
- **コンプリート通知**：ON/OFF・ロール付与ON/OFF
- **スタンプカード画像**：台紙・スタンプ画像・座標の設定
- **コマンド有効/無効**：`/スタンプカード` コマンドの使用可否
