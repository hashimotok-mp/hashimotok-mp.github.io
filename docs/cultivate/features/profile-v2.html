プロフィール v2

カスタマイズ可能なプロフカードとVCインチャ表示の新世代プロフィールシステム

💡 プロフィール v2 とは？ 従来の v1 プロフィールを拡張し、カードデザインの自由なカスタマイズ、 プリセット管理、VCインチャプロフの複数設定を 可能にした新システムです。v1 からの移行機能も備えています。

概要

プロフィール v2 は、サーバーごとに複数のプリセット（設定）を作成し、 それぞれに自由なカードデザイン（Embedのタイトル・説明・色・内容）を割り当てられるシステムです。 また、VC入室時に自動でプロフィールカードを表示するVCインチャプロフも 複数設定を管理でき、VCやカテゴリごとに使用するプリセットを切り替えられます。

📋

プリセット管理

複数のプロフィール設定を保存・切替可能。メイン設定にすれば一元管理も簡単。

🎨

カードデザイン

Embedのタイトル・説明・色・内容を自由にカスタマイズ。プレースホルダーで動的表示。

🔊

VCインチャプロフ

入室時に自動表示、退出時に自動削除。VCやカテゴリごとに設定を割り当て可能。

🔄

v1 移行対応

既存の v1 設定をワンクリックで v2 に移行。v1 は自動無効化。

セットアップ

  1. 設定メニューを開く /プロフィールv2設定 を実行し、設定メニューを表示します。 管理者権限が必要です。
  2. プリセットを作成 「📋 プリセット管理」→「➕ 新規作成」から新しいプリセットを作成します。 名前・対象チャンネル・カードデザインを設定できます。
  3. VCインチャ設定（必要な場合） 「🔊 VCインチャプロフ管理」からVC入室時の自動表示設定を構成します。
  4. v1 からの移行（既存ユーザー） 「🔄 v1から移行」を実行すると、v1 の設定を v2 のプリセットとして取り込みます。

📋 プリセット管理

プリセットはプロフィールの設定のひとかたまりです。 1つのサーバーに複数のプリセットを作成でき、それぞれに異なる カードデザインや対象チャンネルを設定できます。

プリセットの項目

項目 説明
設定名 プリセットの名前（管理用）
状態（有効/無効） プリセットを有効にするかどうか
⭐ メイン設定 ONにすると他のプリセットはすべて自動で無効になります。 1つのサーバーに1つだけ設定できます。
対象チャンネル プロフィールとして自動保存するチャンネルを選択します。 ここで設定したチャンネルに投稿されたメッセージがプロフとして記録されます。
カードデザイン Embedのタイトル・説明・色・内容をカスタマイズします。

メイン設定について

「メイン設定」をONにすると、そのプリセットがそのサーバーの 主要なプロフィール設定として扱われます。 メイン設定をONにした時点で、他のすべてのプリセットは自動的に メイン設定がOFFになります（排他的動作）。

対象チャンネルにメッセージが投稿されたとき、メイン設定がONの プリセットが存在する場合はその設定に従ってプロフィールが保存されます。

🎨 カードデザインのカスタマイズ

プリセットごとにプロフィールカード（Embed）のデザインを自由に設定できます。 編集画面では以下の項目を変更できます。

デザイン項目

項目 説明 デフォルト値
Embedタイトル カード上部に表示されるタイトル {usernickname} のプロフィール
Embed説明 タイトルの下に表示される説明文。 プロフィールURLのリンクを入れるのが一般的です。 {usermention}\n**[プロフィールはコチラ]({profile_url})**
色モード 「固定色」か「ロール基準」を選択。 ロール基準にすると、指定したロールの色がEmbed色として使われます。 固定色（0x00BFFF）
Embed色 固定色モード時の色を16進数カラーコードで指定 #00BFFF
色参照ロール ロール基準モード時に参照するロールを複数選択。 メンバーが持っているロールの中で最上位の色が採用されます。 —
Embed内容 説明文の下に追加で表示するテキスト内容 （空）

使用可能なプレースホルダー

タイトル・説明・内容には以下のプレースホルダーが使用できます。 実際の値に自動で置き換えられて表示されます。

プレースホルダー 置き換え内容
{user} ユーザー名（グローバル名）
{usernickname} サーバーでの表示名（ニックネーム）
{usermention} メンション（@ユーザー）
{guild} サーバー名
{profile_url} プロフィールメッセージへのジャンプURL
{created_at} プロフィールの登録日時
{updated_at} プロフィールの更新日時
{setting_name} 使用中のプリセット名
{newline} 改行

💡 活用例:
{usernickname} のプロフィール | {guild}
と設定すると、「なつき のプロフィール | Cultivate」のように 動的に表示されます。

🔊 VCインチャプロフィール

VC（ボイスチャンネル）に入室したときに、テキストチャンネル（インチャ） に自動でプロフィールカードを表示する機能です。 退出時には自動で削除されます。

VCインチャ設定の項目

項目 説明
設定名 VC設定の名前（管理用）
状態（有効/無効） このVC設定を有効にするかどうか
使用するプリセット このVC設定で使用するプロフィールプリセットを選択（複数可）。 最初に見つかったプロフが表示されます。
有効VC（ホワイトリスト） プロフを表示するVCを限定する場合に指定
除外VC（ブラックリスト） 特定のVCでは表示しない場合に指定
有効カテゴリ 特定のカテゴリ以下のVCのみ対象にする場合に指定
除外カテゴリ 特定のカテゴリ以下のVCを対象外にする場合に指定
Embed内容を表示 プロフカードのEmbed内容（content）を表示するかどうか
プロフなしメッセージ プロフィールが登録されていないメンバーに 「プロフがありません」というメッセージを表示するかどうか

プロフなしメッセージのカスタマイズ

「プロフなしメッセージ」がONの場合、プロフィール未登録のメンバーに 表示するメッセージもカスタマイズできます。 通常のプリセットと同じプレースホルダーが使用可能です。

項目 説明 デフォルト値
タイトル プロフなし時のEmbedタイトル {usernickname} のプロフィール
説明 プロフなし時のEmbed説明 プロフィールが登録されていません。
色 プロフなし時のEmbed色 #E74C3C（赤）
内容 プロフなし時の追加テキスト （空）

⚠️ 注意: VCインチャプロフは VCと同じサーバー内のテキストチャンネル（インスタントチャンネル / インチャ）に表示されます。 インチャが存在しないVCでは機能しません。

🔄 v1 からの移行

既存の v1 プロフィール設定を使用している場合、 管理メニューから「🔄 v1から移行」を実行すると、 v1 の設定を v2 のプリセットとして取り込めます。

移行内容

v1 設定 v2 移行先
プロフィールチャンネル プリセット「v1から移行」の対象チャンネル
VCプロフィール設定 VCインチャ設定「v1から移行(VC)」として移行
Embedデザイン設定 プリセットのカードデザインに反映

💡 移行について: 移行後も v1 の設定データは保持されますが、v1 の VCプロフィール機能は 自動で無効化されます。移行は手動で行う必要があります （サーバー起動時の自動移行は行われません）。

📜 コマンド一覧

管理コマンド

プロフィールv2設定

/プロフィールv2設定

プロフィール v2 の設定メニューを開きます。 プリセット管理・VCインチャ管理・v1移行をこのメニューから行えます。

権限
管理者（Administrator）

ユーザーコマンド

プロフィールv2表示

/プロフィールv2表示 [メンバー]

指定したメンバーのプロフィールを表示します。 メンバーを省略した場合は自分自身のプロフィールが表示されます。

引数 説明
member（省略可） プロフィールを表示するメンバー

🖥️ 設定画面の構成

/プロフィールv2設定 を実行すると、3つのボタンがある メニュー画面が表示されます。

メニューボタン

ボタン 説明
📋 プリセット管理 プロフィールのプリセット一覧を表示します。 各プリセットの編集・新規作成・削除が行えます。
🔊 VCインチャプロフ管理 VCインチャプロフの設定一覧を表示します。 各設定の編集・新規作成・削除が行えます。
🔄 v1から移行 既存の v1 プロフィール設定を v2 に移行します。

プリセット編集画面

プリセット管理で「編集」を押すと、以下の項目を設定できる編集画面が開きます。

基本設定

  • 設定名 — プリセットの名前を変更
  • 有効/無効 — セレクトボックスで切替
  • メイン設定 — セレクトボックスでON/OFF切替（ONにすると他が自動OFF）
  • 対象チャンネル — チャンネルセレクトから複数選択

カードデザイン設定

  • Embedタイトル — テキスト入力（プレースホルダー使用可）
  • Embed説明 — テキスト入力（プレースホルダー使用可）
  • 色モード — 「固定色」または「ロール基準」を選択
  • Embed色 — 固定色モード時の色コード入力
  • 色参照ロール — ロール基準モード時に参照するロールを選択
  • Embed内容 — 追加テキスト（プレースホルダー使用可）

VCインチャ設定の編集画面

VCインチャプロフ管理で「編集」を押すと、以下の項目を設定できます。

  • 設定名 — VC設定の名前
  • 有効/無効 — セレクトボックスで切替
  • 使用するプリセット — プリセット一覧から複数選択
  • 有効VC / 除外VC — VCセレクトから選択
  • 有効カテゴリ / 除外カテゴリ — カテゴリセレクトから選択
  • プロフなしメッセージ表示 — セレクトでON/OFF
  • Embed内容表示 — セレクトでON/OFF
  • プロフなし時のタイトル・説明・色・内容 — 個別にカスタマイズ

⚠️ 注意事項

  • データベース: プロフィール v2 は以下のテーブルを使用します。 手動で編集しないでください。
    • profile_v2cog_configs — プリセット設定
    • profile_v2cog_profiles — 保存されたプロフィールメッセージ
    • profile_v2cog_vc_inchat_configs — VCインチャ設定
  • 互換性: v1 システムとは独立して動作します。 v1 のプロフィールリンクは v2 では使用されません。
  • プロフ再表示: VCインチャプロフが表示されなくなった場合は 一度VCを出入りするか、v1 の /プロフィール再表示 コマンドを 使用してください（v1 が有効な場合）。
