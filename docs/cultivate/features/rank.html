ランクシステム

チャット・VCの活動を計測してレベルアップ

💡 ランクシステムとは？ メッセージ送信やVC接続時間に応じて経験値（XP）を 獲得し、レベルアップしていくシステムです。 サーバーごとに自由に成長曲線を設計できます。

概要

ランクシステムは、ユーザーのDiscord活動量を可視化する機能です。 チャットでの発言やVCへの接続時間に応じてXPが溜まり、 一定量貯まるとレベルアップします。

ミュート勢の利便性を考慮し、VCミュート中もXPが獲得できる設計です。 また、テキストチャットとインチャットを別々に計測する設定も可能です。

機能一覧

機能 説明
経験値計測 メッセージ送信・VC接続時間に応じてXPを付与
レベルアップ 累積XPがしきい値を超えると自動でレベルアップ
ランク付与 特定レベル到達でロールを自動付与（任意設定）
ランキング サーバー内のXPランキングTOP10を表示
チャット分離 TCとインチャットを別々に計測・表示（任意設定）

経験値の獲得方法

チャット経験値

メッセージを送信するとXPが付与されます。 ただし、以下の条件で付与されません。

  • Botユーザーのメッセージ
  • ! や / で始まるコマンド
  • クールダウン期間中の連続送信（デフォルト60秒）

VC経験値

VCに接続している間、一定間隔でXPが付与されます。

  • 1分ごとにXPを獲得
  • 2人以上が接続している場合のみ有効
  • ミュート中でもXPが獲得可能
  • 1人VCでは獲得できません

⚠️ 注意: VCから切断すると、接続開始時からの経過時間に応じたXPが 一括で計算されて加算されます。

レベル・経験値の仕組み

デフォルトの成長曲線

𝓒𝓾𝓵𝓽𝓲𝓿𝓪𝓽𝓮標準の成長曲線は二次関数を採用しています。 

次レベル必要XP = 5 × L² + 50 × L + 100

（L = 現在のレベル）

レベル 次レベルまで 累積XP
Lv.1 → Lv.2 155 XP 155
Lv.5 → Lv.6 280 XP 1,130
Lv.10 → Lv.11 680 XP 4,355
Lv.50 → Lv.51 12,800 XP 430,550

高度設定：カスタム数式

管理者は任意の数式で成長曲線を設計できます。 二次関数だけでなく、線形制や緩やかな曲線など自由に設定可能です。

使用可能な構文

要素 説明 例
L 現在のレベル 5 * L + 100
+ - * / // % ** 四則演算・累乗 L ** 2
min max abs 基本関数 max(100, 50 * L)
pow int float round 累乗・型変換・丸め pow(L, 2)

設定例

5 * L ** 2 + 50 * L + 100     # デフォルト（二次関数）
100 * L + 50                     # 線形制（ゲームバランス向け）
max(100, 50 * L ** 1.5)          # 緩やか→急激（イベント向け）

💡 累積XP式について: 累積XPの計算式は通常、次レベルXP式から自動導出されます。 高度な用途で独自に設定したい場合のみ、累積XP式も入力できます。

ランクロール設定

特定のレベルに到達した際に、自動でロールを付与できます。 例えば「Lv.10到達で『ベテラン』ロールを付与」などの設定が可能です。

注意事項

  • BotユーザーのメッセージはXP計測の対象外です
  • 同じメッセージの連投はクールダウンで制限されます
  • VCは2人以上で接続している場合のみXPが付与されます
  • レベルアップ通知は設定したチャンネルに送信されます
  • カスタム数式の結果が0以下になった場合、デフォルト式に戻ります
