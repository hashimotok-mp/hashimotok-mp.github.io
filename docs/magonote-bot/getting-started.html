🚀 導入方法

まごの手 をサーバーに導入する手順を説明します。

必要環境

  • Python 3.10 以上
  • discord.py 2.3 以上
  • サーバーの管理者権限

導入手順

  1. リポジトリをクローン 
    git clone https://github.com/hashimotok-mp/magonote-bot.git
cd magonote-bot
  2. 依存パッケージをインストール 
    pip install -r requirements.txt
  3. 設定ファイルを準備 config.yml を編集し、各機能の設定を行います。 設定項目の詳細は各機能のドキュメントを参照してください。
  4. Bot トークンを設定 SecretVar.py に Discord Developer Portal で発行した Bot トークンを設定します。
  5. Bot を起動 
    python magonote_main.py

💡 注意: Bot の稼働には Python 環境と Bot トークンが必要です。自己ホスト型の Bot ですので、稼働サーバーは別途ご用意ください。

初期設定

Bot が起動したら、以下の設定を行ってください。

config.yml の編集

config.yml で各機能の設定を行います。主な設定項目は以下の通りです。

設定項目 説明
nagara ながらバッチのボタン設定
emoji 自動リアクションの設定
keyword_role キーワードロールの設定
random_msg ランダム転載チャンネルの設定
role_filter ロールフィルターの設定
template 定型文テンプレートの設定
auto_channel_message 自動チャンネルメッセージの設定
vc_edit_panel VC編集パネルのカテゴリ設定
vc_connect_auto_permission VC接続時権限自動追加の設定
role_mention_setting ロールメンション昼夜切替の設定
vc_start_notice VC開始通知の設定
vc_permission_setting VC権限昼夜切替の設定

スラッシュコマンドの同期

Bot 起動後、自動的にスラッシュコマンドが同期されます。 反映には数分かかる場合があります。管理者は /sync コマンドで手動同期も可能です。
