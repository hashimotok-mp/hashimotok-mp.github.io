ランダム転載

過去のメッセージをランダムに掘り起こし

概要

ランダム転載機能は、指定したチャンネルからランダムにメッセージを選び、 現在のチャンネルに転載表示する機能です。過去の面白い会話や有益な情報を 再発見するのに役立ちます。

設定

config.yml の random_msg セクションで、 ランダム転載の対象チャンネルを設定します。 

random_msg:
  0:
    channel: チャンネルID
    mask_id: []

使い方

/ランダム コマンドを実行し、対象チャンネルを選択します。 設定ファイルで指定したチャンネルのみが選択肢に表示されます。

💡 ヒント: 転載されるメッセージには元のメッセージへのジャンプリンクが付与されます。
