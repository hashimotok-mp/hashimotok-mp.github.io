ノックパネル

VC への安全な入室リクエスト

概要

ノックパネル機能は、VC に入室したいメンバーが「ノック（入室リクエスト）」を送信し、 相手の許可を得てから入室できる仕組みを提供します。 プライベートな VC でのコミュニケーションを安全に行うために使用します。

使い方

パネルの設置

管理者が /ノックパネル作成 コマンドを実行すると、ノックパネルが設置されます。

ノックする

  1. パネルの「ノック」ボタンを押す
  2. 入りたい VC に接続しているメンバーを選択
  3. Bot から相手に DM でノック通知が届く
  4. 相手が DM 内の「許可」ボタンを押すと入室できる

招待する

「招待」ボタンを使うと、相手がノックする前に招待を送ることもできます。

💡 ヒント: ノック機能は、特定の VC に参加したいが、相手に直接声をかけづらい場合などに便利です。
