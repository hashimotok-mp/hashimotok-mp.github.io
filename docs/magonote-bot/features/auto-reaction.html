自動リアクション

メッセージ投稿時に自動でリアクション

概要

自動リアクション機能は、特定のチャンネルにメッセージが投稿されたときに、 自動で指定された絵文字リアクションを追加する機能です。 コミュニケーションの活性化や、特定の投稿への反応を自動化できます。

設定

config.yml の emoji セクションで設定します。 

emoji:
  0:
    guild_id: サーバーID
    channel_id: チャンネルID
    bot: "false"
    reactions:
      - "👍"
      - "❤️"
      - "😊"
項目 説明
guild_id サーバーID
channel_id 自動リアクションを付けるチャンネルID
bot Botの投稿にもリアクションを付けるか（"true"/"false"）
reactions 追加するリアクションの一覧（絵文字またはカスタム絵文字ID）

💡 ヒント: カスタム絵文字を使用する場合は、絵文字ID（数値）を指定します。Unicode絵文字はそのまま文字列で指定可能です。
