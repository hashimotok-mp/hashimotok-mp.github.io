キーワードロール

投稿内容に応じて自動でロールを付与・剥奪

概要

キーワードロール機能は、特定のチャンネルで特定のキーワードを含むメッセージを 投稿したメンバーに対して、自動でロールの付与・剥奪を行う機能です。 例として、自己紹介チャンネルで「名前」と投稿すると自動でロールが付与される、 といった使い方ができます。

設定

config.yml の keyword_role セクションで設定します。 

keyword_role:
  0:
    guild_id: サーバーID
    channel_id: チャンネルID
    keywords:
      - "キーワード"
    add_roles:
      - 付与ロールID
    remove_roles:
      - 剥奪ロールID
項目 説明
guild_id サーバーID
channel_id 監視するチャンネルID
keywords トリガーとなるキーワード一覧（AND条件）。空の場合は全メッセージが対象
add_roles 条件一致時に付与するロールID一覧
remove_roles 条件一致時に剥奪するロールID一覧

⚠ 注意: キーワードは AND 条件で判定されます。複数のキーワードを設定した場合、全てのキーワードがメッセージに含まれている必要があります。
