ロールメンション切替

昼夜の特定時刻にロールのメンション可否を自動切替

概要

ロールメンション切替機能は、特定の時刻になるとロールのメンション可否を 自動で切り替える機能です。例えば、昼の12時にあるロールのメンションを有効にし、 夜の24時に無効にする、といった設定が可能です。

設定

config.yml の role_mention_setting セクションで設定します。 

role_mention_setting:
  サーバーID:
    noon_time: "12:00"
    night_time: "24:00"
    noon_role:
      - 昼にメンション可能にするロールID
    night_role:
      - 夜にメンション可能にするロールID
項目 説明
サーバーID サーバーID
noon_time 昼の切替時刻（例: "12:00"）
night_time 夜の切替時刻（例: "24:00"）
noon_role 昼にメンション可能にするロールIDのリスト
night_role 夜にメンション可能にするロールIDのリスト

💡 ヒント: この機能は1分ごとに実行され、時刻が設定値と一致した場合にロールのメンション設定を切り替えます。
