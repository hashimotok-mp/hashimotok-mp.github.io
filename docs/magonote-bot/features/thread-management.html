スレッド管理

スレッド作成時に自動で管理UIを設置

概要

スレッド管理機能は、特定のチャンネルでスレッドが作成されたときに、 自動的に削除・名称変更・招待の3つのボタンを設置する機能です。 スレッドの管理を簡素化し、運営の手間を削減します。

設定

config.yml の thread_channels セクションで、 管理対象とするチャンネルIDをリストで指定します。 

thread_channels:
  - チャンネルID1
  - チャンネルID2
  - チャンネルID3

自動設置されるボタン

ボタン 機能
削除 スレッドを削除します
名称変更 スレッド名を変更します
招待 メンバーをスレッドに招待します

💡 ヒント: ボタンはスレッドの最初のメッセージとしてピン留めされた状態で投稿されます。

自動チャンネルメッセージ

スレッド作成時に、設定に基づいて自動メッセージを投稿することもできます。 詳細は 自動チャンネルメッセージ のページを参照してください。
