定型文テンプレート

コンテキストメニューからワンクリックで定型文を送信

概要

定型文テンプレート機能は、ユーザーを右クリック → アプリ → 定型文テンプレート のコンテキストメニューから、事前に設定した定型文を DM で送信できる機能です。 定型文は設定ファイルで自由にカスタマイズでき、複数のテンプレートを サーバーごとに設定できます。

設定

config.yml の template セクションで設定します。 

template:
  0:
    menu_title: "テンプレート名"
    setting:
      サーバーID:
        message_link: "https://discord.com/channels/..."
        templates:
          - "定型文1"
          - "定型文2"
項目 説明
menu_title コンテキストメニューに表示される名前
message_link テンプレートの説明が書かれたメッセージへのリンク
templates 送信する定型文のリスト（Embed形式で送信）

使い方

  1. サーバー内の任意のユーザーを右クリック
  2. 「アプリ」→ テンプレート名 を選択
  3. 送信者と受信者の両方に確認のDMが送信される

💡 ヒント: 挨拶文や注意文、ウェルカムメッセージなど、よく使う文章をテンプレート化しておくと便利です。
