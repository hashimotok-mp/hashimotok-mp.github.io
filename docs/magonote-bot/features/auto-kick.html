自動キック

指定ロールを持たないメンバーを一括キック

概要

自動キック機能は、指定したロールのいずれも保持していないメンバーを サーバーから一括でキックするコマンドです。サーバーの整理や、 特定の条件を満たさないメンバーの一斉退出処理に使用します。

使い方

/自動キック role1:ロール1 [role2:ロール2] ... [role10:ロール10]

必須の role1 に加えて、最大9つの追加ロール（role2〜role10）を任意で指定できます。 指定したいずれのロールも持っていないメンバーがキック対象となります。

引数 必須 説明
role1 必須 保持を確認する基準ロール1
role2〜role10 任意 追加の基準ロール

⚠ 注意: このコマンドはサーバーの管理者権限が必要です。実行前に確認ダイアログが表示されます。キックされたメンバーを復元するには、サーバー管理者が手動で再招待する必要があります。
