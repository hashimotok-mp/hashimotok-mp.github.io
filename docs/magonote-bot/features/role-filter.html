ロールフィルター

性別ロールなどをフィルタリングするパネル

概要

ロールフィルター機能は、性別ロール（例: 本男 / 本女）などのベースロールを 選択・フィルタリングするためのパネルを設置できる機能です。 特定のロールをベースに、サーバーのメンバーをフィルタリングできます。

設定

config.yml の role_filter セクションで設定します。 

role_filter:
  0:
    guild: サーバーID
    name: "フィルター名"
    description: "パネルの説明文"
    base_role1: ロールID1
    base_role2: ロールID2
    button_name1: "ボタン名1"
    button_name2: "ボタン名2"
    search_channel1: 検索チャンネルID1
    search_channel2: 検索チャンネルID2
    filter_roles:
      - フィルターロールID
項目 説明
guild サーバーID
name パネル名（オートコンプリートに表示）
description パネルの説明文
base_role1/2 フィルタリングの基準となるロール（例: 性別ロール）
button_name1/2 パネルに表示するボタン名
search_channel1/2 メンバーを検索するチャンネル
filter_roles フィルタリング対象のロール一覧

使い方

  1. /ロールフィルターパネル作成 コマンドを実行
  2. パネル名を選択（設定ファイルで定義した名前が候補に表示）
  3. 表示されたパネルのボタンでフィルタリングを実行
