VC権限昼夜切替

時刻に応じてVCやチャンネルの権限を自動切替

概要

VC権限昼夜切替機能は、設定された時刻に応じて、VCやチャンネルの 権限設定を自動で切り替える機能です。昼用と夜用の権限設定をそれぞれ定義し、 指定時刻になると自動で権限が変更されます。

設定

config.yml の vc_permission_setting セクションで設定します。 

vc_permission_setting:
  サーバーID:
    VCまたはチャンネルID:
      noon_times:
        - "08:00"
      night_times:
        - "22:00"
      noon_permission:
        view_channel: true
        connect: true
        send_message: true
        send_thread: true
      night_permission:
        view_channel: false
        connect: false
        send_message: false
        send_thread: false
項目 説明
noon_times 昼の権限に切り替える時刻（複数設定可能）
night_times 夜の権限に切り替える時刻（複数設定可能）
noon_permission 昼間の権限設定
night_permission 夜間の権限設定
