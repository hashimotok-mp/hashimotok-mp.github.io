VC操作パネル

ボタン一つで VC の各種設定を操作

概要

VC操作パネルは、自分が接続している VC のルーム名・人数制限・ステータス・ビットレートなどを ボタン操作で簡単に変更できるパネルです。ルームに接続している全員が操作できます。

使い方

パネルの表示

/vc操作パネル作成 コマンドを実行すると、VC操作パネルが表示されます。

操作ボタン

ボタン 機能
ルーム名 VC のルーム名を変更
人数 VC の最大人数を変更
ステータス VC のステータスを変更
ビットレート VC のビットレートを変更
全体から隠す VC を全員から見えないようにする
個人から隠す 特定のメンバーから VC を隠す
招待 メンバーを VC に招待

設定

config.yml の vc_edit_panel セクションで、 操作パネルを表示するカテゴリと権限ロールを設定できます。 

vc_edit_panel:
  サーバーID:
    categories:
      - カテゴリID
    roles:
      - ロールID
