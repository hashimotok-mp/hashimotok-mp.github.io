リアクション計測

メッセージのリアクションを集計・ランキング表示

概要

リアクション計測機能は、指定したチャンネルのメッセージに対するリアクションを 集計し、ランキング形式で表示する機能です。全リアクションまたは指定した絵文字の リアクションのみを対象に、上位ランキングを表示できます。

使い方

コマンド

/リアクション計測 channel:チャンネル type:全リアクション|指定リアクション count:上位件数 [emoji:絵文字]
引数 必須 説明
channel 必須 計測対象のチャンネル
type 必須 "全リアクション" または "指定リアクション"
count 必須 表示する上位件数
emoji 任意 指定リアクションの場合に計測する絵文字

⚠ 注意: 大量のメッセージをスキャンする場合、処理に時間がかかることがあります。
