VC接続時権限自動追加

VC 参加時に自動で権限を付与

概要

VC接続時権限自動追加機能は、メンバーが VC に参加したときに、 自動的にその VC の管理権限（connect, view_channel, manage_channels, speak, send_messages）を付与する機能です。 これにより、VC を作成したメンバーがルームの管理をしやすくなります。

設定

config.yml の vc_connect_auto_permission セクションで設定します。 

vc_connect_auto_permission:
  サーバーID:
    カテゴリID:
      disable_vc:
        - 除外するVCチャンネルID
項目 説明
サーバーID サーバーID
カテゴリID 対象とするカテゴリID
disable_vc 権限自動追加を除外するVCチャンネルIDのリスト

💡 ヒント: 特定のVC（例: 待機所など）では権限自動追加を無効にしたい場合、disable_vc リストにそのVCのIDを追加してください。
