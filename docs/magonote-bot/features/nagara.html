ながらバッチ

VC で「何をしているか」をニックネームで共有

概要

ながらバッチ機能は、VC に参加中に自分の活動状態をサーバーニックネームの先頭に表示できる機能です。 「作業中」「ゲーム中」「動画視聴中」など、ボタンを押すだけで簡単に状態を切り替えられます。

使い方

1. パネルを表示

/ながら コマンドを実行すると、ながらバッチパネルが表示されます。

2. 状態を選択

パネルに表示されたボタン（例: 「作業」「ゲーム」「動画」など）を押すと、 サーバーニックネームの先頭にその状態を示す文字が追加されます。

3. リセット

「サーバーネーム」ボタンでニックネームをリセットできます。 「表示名」ボタンでDiscordの表示名に戻せます。

設定

config.yml の nagara セクションで設定します。 

nagara:
  サーバーID:
    buttons:
      - "作業"
      - "ゲーム"
      - "動画"
      - "麻雀"
      - "仕事"
      - "家事"
      - "食事"
      - "運転"
      - "外出"
      - "子育て"
      - "猥談"
      - "🔞️"
      - "眠気"
      - "フラグ"
      - "ミュート"
      - "静か"
項目 説明
サーバーID Discord サーバーの ID
buttons 表示するボタンのラベル一覧。最大25個まで。

💡 ヒント: サーバーごとに異なるボタン設定が可能です。サーバーの雰囲気に合わせてカスタマイズしてください。
