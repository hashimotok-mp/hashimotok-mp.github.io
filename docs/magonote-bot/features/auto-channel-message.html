自動チャンネルメッセージ

チャンネル/スレッド作成時に自動でメッセージを投稿

概要

自動チャンネルメッセージ機能は、新しいチャンネル（テキスト/VC）や スレッドが作成されたときに、自動で設定されたメッセージを投稿する機能です。 ルール説明や注意事項などを自動掲載することで、運営の手間を削減します。

設定

config.yml の auto_channel_message セクションで設定します。 

auto_channel_message:
  0:
    guild: サーバーID
    category: true
    id: カテゴリIDまたはチャンネルID
    vcpanel: false
    notes:
      content: null
      embed: null
    messages:
      0:
        content: "メッセージ本文"
        embed: "埋め込みテキスト"
項目 説明
guild サーバーID
category true の場合はカテゴリ内のチャンネル作成時、false の場合は特定チャンネルのスレッド作成時に発動
id カテゴリID または チャンネルID
vcpanel VC作成時にVC操作パネルも自動設置するか
messages 自動投稿するメッセージのリスト（content/embed を指定）
notes メッセージ投稿後に追加で送信するノート

💡 ヒント: category を true にすると、そのカテゴリ内に新しいチャンネルが作成されるたびに自動メッセージが投稿されます。false の場合は、そのチャンネル内にスレッドが作成されたときに発動します。
