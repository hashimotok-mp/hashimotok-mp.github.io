📜 コマンド一覧

まごの手 で使用できるスラッシュコマンドの一覧です。

VC系コマンド

/ながら
ながらバッチパネルを表示。VCでの活動状態をニックネームに設定できます。
一般
/満室
自分が接続しているVCのルーム名の先頭に 🈵 を付けます。
一般
/満室解除
自分が接続しているVCのルーム名から 🈵 を外します。
一般
/ノックパネル作成
VCへのノック（入室リクエスト）パネルを作成します。
管理者
/vc操作パネル作成
VCのルーム名・人数制限・ステータス・ビットレートなどを操作できるパネルを作成します。
一般
/vcリージョン変更設定
VCのリージョンを定期的に変更するスケジュールを管理するパネルを表示します。
管理者

ロール管理コマンド

/取得ロール確認パネル
自分に付与されているロール一覧を表示するパネルを作成します。
管理者
/ロールフィルターパネル作成
性別ロールなどをフィルタリングするパネルを作成します。パネル名は設定ファイルで定義。
管理者
/chリアクションロール付与
指定チャンネルのメッセージにリアクションを付けたユーザーにロールを付与します。
管理者 引数: channel / role / emoji(任意)
/ロールメンバー
指定した条件に合致するメンバー一覧を表示します。タイプ: Aが付いている / Aが付いていない / AB両方付いてる / AB両方付いていない / Aの中でBが付いていない
管理者 引数: type / role_a / role_b(任意)

メッセージ管理コマンド

/ランダム
指定チャンネルからランダムにメッセージを転載して表示します。
一般 引数: チャンネル選択
/リアクション計測
指定チャンネルのメッセージリアクションを集計し、ランキングを表示します。
管理者 引数: channel / type / count / emoji(任意)
/purge
指定した2つのメッセージIDの間のメッセージを一括削除します。
管理者 引数: channel / message_id1 / message_id2

管理コマンド

/bot監視設定
24時間稼働Botのメッセージを監視し、停止を検知して通知する設定パネルを表示します。
管理者
/サーバーカウンター設定
VC名を自動更新するカウンター設定を管理するパネルを表示します。
管理者
/自動キック
指定したロールを持たないメンバーを一括キックします。最大10ロールまで指定可能。
管理者 引数: role1 ~ role10
/カテメンバーパネル作成
カテゴリごとにメンバーの閲覧・接続権限を管理するパネルを作成します。
管理者

コンテキストメニュー

定型文テンプレート（コンテキストメニュー）
ユーザーを右クリック → アプリ → 定型文テンプレート で定型文をDM送信できます。設定ファイルで自由にカスタマイズ可能。
管理者
